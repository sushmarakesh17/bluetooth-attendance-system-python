# Bluetooth Attendance System Using Python

## Project Overview

Bluetooth Attendance System is a Python automation project that enables automated attendance tracking using Bluetooth device detection.

The system identifies registered Bluetooth devices and records attendance information, reducing manual attendance processes.

## Objectives

- Automate attendance tracking
- Detect registered Bluetooth devices
- Maintain attendance records digitally
- Reduce manual attendance effort

## Technologies Used

- Python
- Bluetooth Libraries
- CSV / Database Storage
- Automation Scripting

## Features

- Bluetooth device scanning
- Device identification
- Automatic attendance marking
- Attendance record management
- Digital attendance tracking

## Project Workflow

1. Scan available Bluetooth devices
2. Identify registered devices
3. Record attendance details
4. Store attendance data
5. Generate attendance records
